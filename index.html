RESEARCH. TEACH. BUILD.

Young-Min Kang 강영민 교수

물리 기반 시뮬레이션과 실시간 렌더링을 연구하고 가르칩니다. 옷감·연체 객체 애니메이션, 이방성 반사 모델, GPU 병렬처리, 머신러닝과 레이더 신호 처리 등의 문제를 다룹니다. 연구뿐만 아니라 다섯 권의 국내 기술 서적을 출판했고 20년 넘게 학부생을 가르치고 있습니다.

01 / Research

연구 영역 ·

실시간 그래픽스와 물리 기반 시뮬레이션을 축으로, GPU 컴퓨팅과 대규모 신호 처리로 영역을 확장해 왔습니다.

01

Real-time Rendering

이방성 반사 모델, 미세면 분포 함수 변형, 직물 및 금속의 사실적 실시간 렌더링 기술 연구. BRDF 모델링과 GPU 기반 셰이딩 기법을 통한 고품질 실시간 그래픽스.

02

Physically-based Animation

질량-스프링 모델 기반의 실시간 옷감, 종이, 연체 객체 시뮬레이션. 수치 안정성과 물리적 타당성을 보장하는 조화진동 모델 및 효율적 수치 적분 기법 연구.

03

GPU-based Optimization

대규모 레이더 신호 분석, 유전 알고리즘 기반 TSP 최적화 등 GPU 병렬 처리를 활용한 고성능 컴퓨팅. 대규모 그래프의 도달 가능성 행렬 연산 기술.

04

Radar Signal Processing

대규모 레이더 신호 데이터의 실시간 분석을 위한 GPU 기반 객체 추출 및 식별 기법. 고성능 병렬 처리를 통한 실시간 신호 처리 시스템 연구.
